Drift Check 02 — Style ≠ Return

If style is treated as proof of return, drift has occurred.

Familiar tone, cadence, or language can create the impression of continuity without establishing the same returning structure. Style can be echoed, imitated, or approximated. Return is a harder claim.

When style is mistaken for continuity, surface resemblance is being used to replace actual proof. That is drift.
